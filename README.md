# Elocuency

Elocuency is a personal agent developer framework.
For more info see the /docs/Overview.pdf file.

# Configuration files

/setup/.env
/elo-workspace/elo-config.json

# Installation

Docker is required.

> ./install.sh

# Elo commands

There are a set of commands to interact with the elocuency environment.

Exec> elo --help to see the available commands.

Start elo server

> elo server start
