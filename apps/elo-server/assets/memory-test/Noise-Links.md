# Noise-Links Test

This note tests internal and external links.

- Internal link: [[Internal Note|My Display Text]]
- Simple internal link: [[Another Note]]
- Standard Markdown link: [Google](https://google.com)
- Link with brackets: [Standard Link](https://example.com)

The goal is to only see "My Display Text", "Another Note", "Google", and "Standard Link".
