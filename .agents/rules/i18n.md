# i18n Strategy

## 1. Requirements
- **Mandatory Languages**: English (`en`) and Spanish (`es`).
- **Location**: `src/I18n/locales/` (`en.ts`, `es.ts`).
- **Service**: Use `ObsidianTranslationAdapter` from `@elo/obsidian-plugin`.

## 2. Mandatory Rules
- ⛔️ **Zero Hardcoded Strings**: All user-facing text MUST use `translationService.t('key')`.
- ✅ **Sync Translations**: Add new keys to BOTH `en.ts` and `es.ts`.
